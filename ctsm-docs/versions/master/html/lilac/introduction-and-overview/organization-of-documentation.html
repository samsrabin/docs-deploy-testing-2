3.1.2. Organization of the documentation

This documentation is organized into the following high-level sections:

  • 3.1. Introduction and overview of LILAC: This section gives a general introduction to LILAC and describes the organization of this documentation (you’re reading this now!)

  • 3.2. Obtaining, building and running CTSM with LILAC: This section provides instructions for building and running CTSM within an atmosphere model that has been set up to run with CTSM (e.g., WRF). If you are starting to use CTSM with an atmosphere model that does not yet have any calls to CTSM-LILAC, then you should start with section 3.3. Calling CTSM from an atmosphere model.

  • 3.3. Calling CTSM from an atmosphere model: This section provides details on the Fortran code that needs to be added to an atmosphere model in order to call CTSM via LILAC as its land surface scheme. (In practice, this step comes before 3.2. Obtaining, building and running CTSM with LILAC, but it is included later in the documentation because it is of interest to fewer people.)

  • 3.4. Instructions on using CTSM with specific atmosphere models: This section provides notes on running CTSM within specific atmosphere models.
